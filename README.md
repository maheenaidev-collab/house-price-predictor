# 🏠 House Price Predictor

A Machine Learning model that predicts house prices based on features like size, bedrooms, bathrooms, and age.

## Features
- Linear Regression model
- Synthetic dataset generation
- Model evaluation (RMSE, R² Score)
- Custom price predictions

## Tech Stack
- Python 3.10+
- NumPy, Pandas, scikit-learn

## Installation
1. Clone the repo
2. Install: `pip install -r requirements.txt`
3. Run: `python predictor.py`
